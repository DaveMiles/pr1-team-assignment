PROJECT TITLE: Medieval Dungeon
PURPOSE OF PROJECT: COM4023 PR1 Part B
VERSION or DATE: 1.0
HOW TO START THIS PROJECT: This is a Greenfoot game and can be imported and run directly in the Greenfoot application.
AUTHORS: David Miles, Fraser Aryes
USER INSTRUCTIONS: 
Your goal is to make your way through the dungeon, past enemies and across platforms to reach the exit.
If you die, you must restart the game.

CONTROLS:
Left / Right - allow the user to move left or right
Up - allow the user to jump over obstacles
Space - attack enemies
F - interact with consumables (Potions, etc).
